# 梦境记录与分享平台 - 分步项目指导

本文档将一步步指导你完成软件工程课程项目。

---

## 整体流程概览

```
第一步：项目立项与需求确认  →  第二步：用例模型  →  第三步：设计模型  →  第四步：系统实现与验收
     （与老师确认）              （作业1，35%）        （作业2，35%）         （作业3，30%）
```

---

## 第一步：项目立项与需求确认

**目标**：明确项目范围，列出功能点，形成可提交给老师确认的选题说明。

**产出物**：
- 项目选题说明文档
- 功能需求清单（≥10个，不含注册登录）
- 老师确认后的最终选题

**详见**：`01-requirements/` 目录

---

## 第二步：用例模型（作业1）

**目标**：识别系统参与者与用例，绘制用例图。

**产出物**：
- 参与者列表
- 用例列表及描述
- 用例图（UML）

**详见**：`02-use-case-model/` 目录

---

## 第三步：设计模型（作业2）

**目标**：完成静态结构设计（类图）和动态结构设计（时序图、活动图等）。

**产出物**：
- 静态结构：类图、包图等
- 动态结构：时序图、活动图、状态图等

**详见**：`03-design-model/` 目录

---

## 第四步：系统实现与验收（作业3）

**目标**：实现系统，通过验收。

**产出物**：
- 可运行的系统
- 部署说明、使用说明

**详见**：`04-system/` 目录
