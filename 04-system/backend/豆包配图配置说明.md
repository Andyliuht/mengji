# 豆包 Seedream 文生图配置说明

## 1. 开通服务

1. 打开 [火山方舟控制台](https://console.volcengine.com/ark)
2. 登录/注册火山引擎账号
3. 进入 **模型推理** → **推理接入点** → **创建推理接入点**
4. 选择 **图片生成** 类目，模型选择 **Doubao-Seedream-4.5** 或 **Doubao-Seedream-4.0**
5. 创建完成后，记下 **接入点 ID**（格式如 `ep-20240920xxxxxx`）
6. 在 **API Key 管理** 中创建并复制 API Key

## 2. 配置 .env

在 `04-system/backend` 目录的 `.env` 中添加：

```
ARK_API_KEY=你的API_Key
SEEDREAM_MODEL=你的接入点ID
```

例如：
```
ARK_API_KEY=sk-xxxxxxxx
SEEDREAM_MODEL=ep-20240920abcdef
```

## 3. 免费额度

- **Doubao-Seedream-4.5**：200 张/免费额度
- **Doubao-Seedream-4.0**：200 张/免费额度

具体以火山方舟控制台显示为准。

## 4. 使用

在梦境详情页点击「AI 生成配图」，系统会根据梦境内容调用豆包生成一张梦幻风格插画，并保存为梦境配图。
