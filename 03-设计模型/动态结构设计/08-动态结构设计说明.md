# 动态结构设计说明

## 一、时序图（Sequence Diagram）

| 文件 | 对应用例 | 说明 |
|------|----------|------|
| `01-时序图-记录梦境.puml` | UC03 记录梦境 | 用户提交梦境 → Controller → Service → TagService → 数据库 |
| `02-时序图-获取AI解梦.puml` | UC07 获取AI解梦 | 用户请求解梦 → 关键词提取 → 规则匹配 → 生成/读取解梦结果 |
| `03-时序图-相似梦境匹配.puml` | UC09 相似梦境匹配 | 用户请求匹配 → 查询当前梦境 → 计算与已分享梦境的相似度 → 返回结果 |
| `04-时序图-分享梦境.puml` | UC10 分享梦境 | 用户选择分享 → 创建 SharedDream → 更新梦境状态 |

## 二、活动图（Activity Diagram）

| 文件 | 说明 |
|------|------|
| `05-活动图-记录梦境.puml` | 记录梦境的用户操作流程，含分支（内容为空、保存失败） |
| `06-活动图-AI解梦.puml` | AI解梦的流程，含缓存判断、收藏分支 |

## 三、状态图（State Diagram）

| 文件 | 说明 |
|------|------|
| `07-状态图-梦境.puml` | 梦境记录的状态流转：草稿 → 已保存 → 已分享，及删除/取消分享 |

## 四、查看与导出

- 使用 [PlantUML 在线编辑器](https://www.plantuml.com/plantuml/uml/) 粘贴 `.puml` 文件内容生成图片
- 或使用 VS Code PlantUML 插件、命令行 `java -jar plantuml.jar *.puml -tpng` 批量导出
