## 二、用例详细描述（核心用例）

### UC03 记录梦境


| 项目       | 内容                                                                                  |
| -------- | ----------------------------------------------------------------------------------- |
| **用例名称** | 记录梦境                                                                                |
| **参与者**  | 注册用户                                                                                |
| **前置条件** | 用户已登录                                                                               |
| **后置条件** | 梦境记录已保存，可被后续功能使用                                                                    |
| **基本流程** | 1. 用户选择「记录梦境」 2. 用户输入梦境内容（支持文字、语音、图片） 3. 用户选择或添加标签、情绪、睡眠质量 4. 用户提交保存 5. 系统保存梦境并提示成功 |
| **扩展流程** | 3a. 用户可选择跳过标签/情绪，使用默认值 5a. 保存失败时，系统提示错误原因                                           |
| **业务规则** | 梦境内容不能为空；单条记录有字数/大小限制                                                               |

### UC07 获取AI解梦


| 项目       | 内容                                                                          |
| -------- | --------------------------------------------------------------------------- |
| **用例名称** | 获取AI解梦                                                                      |
| **参与者**  | 注册用户                                                                        |
| **前置条件** | 用户已登录，且存在梦境记录                                                               |
| **后置条件** | 生成解梦结果，可供用户查看和收藏                                                            |
| **基本流程** | 1. 用户选择一条梦境记录 2. 用户点击「AI解梦」 3. 系统分析梦境内容（关键词、主题等） 4. 系统生成解梦解读 5. 系统展示解梦结果给用户 |
| **扩展流程** | 3a. 分析失败时，提示用户稍后重试 5a. 用户可收藏解梦结果                                            |
| **业务规则** | 解梦基于规则/关键词或AI模型；结果仅供参考                                                      |


---

### UC09 相似梦境匹配


| 项目       | 内容                                                                             |
| -------- | ------------------------------------------------------------------------------ |
| **用例名称** | 相似梦境匹配                                                                         |
| **参与者**  | 注册用户                                                                           |
| **前置条件** | 用户已登录，且存在梦境记录                                                                  |
| **后置条件** | 展示相似梦境列表                                                                       |
| **基本流程** | 1. 用户选择一条梦境记录 2. 用户点击「相似梦境」 3. 系统根据标签、关键词计算相似度 4. 系统返回相似梦境列表（匿名展示） 5. 用户浏览匹配结果 |
| **扩展流程** | 4a. 无相似梦境时，提示「暂无相似梦境」                                                          |
| **业务规则** | 仅匹配已分享的梦境；展示时匿名处理                                                              |

### UC10 分享梦境


| 项目       | 内容                                                                   |
| -------- | -------------------------------------------------------------------- |
| **用例名称** | 分享梦境                                                                 |
| **参与者**  | 注册用户                                                                 |
| **前置条件** | 用户已登录，且存在梦境记录                                                        |
| **后置条件** | 梦境在社区中可见（匿名或公开）                                                      |
| **基本流程** | 1. 用户选择一条梦境记录 2. 用户点击「分享」 3. 用户选择分享方式（匿名/公开） 4. 用户确认分享 5. 系统将梦境发布到社区 |
| **扩展流程** | 4a. 用户可编辑分享时的可见标签 5a. 分享后用户可取消分享                                     |
| **业务规则** | 匿名分享不展示用户身份；公开分享可展示昵称                                                |

### UC18 查看个人梦境统计


| 项目       | 内容                                                                  |
| -------- | ------------------------------------------------------------------- |
| **用例名称** | 查看个人梦境统计                                                            |
| **参与者**  | 注册用户                                                                |
| **前置条件** | 用户已登录，且存在梦境记录                                                       |
| **后置条件** | 无                                                                   |
| **基本流程** | 1. 用户进入「我的统计」 2. 系统汇总用户梦境数据 3. 系统展示主题分布、情绪趋势、记录频率等图表 4. 用户可切换时间范围查看 |
| **扩展流程** | 2a. 无记录时，提示用户先记录梦境                                                  |
| **业务规则** | 统计仅基于用户个人数据；支持按周/月/年筛选                                              |

### UC19 导出梦境


| 项目       | 内容                                                                                   |
| -------- | ------------------------------------------------------------------------------------ |
| **用例名称** | 导出梦境                                                                                 |
| **参与者**  | 注册用户                                                                                 |
| **前置条件** | 用户已登录，且存在梦境记录                                                                        |
| **后置条件** | 生成导出文件供用户下载                                                                          |
| **基本流程** | 1. 用户选择「导出梦境」 2. 用户选择导出范围（全部/时间范围/选中记录） 3. 用户选择导出格式（PDF/文本） 4. 用户确认导出 5. 系统生成文件并提供下载 |

| **业务规则** | 单次导出有数量限制；PDF可包含解梦结果                                                                 |

## 三、用例之间的关系

### 包含关系（Include）

- **记录梦境** 包含 **管理梦境标签**（记录时需选择/添加标签）

### 扩展关系（Extend）

- **浏览分享梦境** 可扩展 **评论梦境**、**点赞梦境**、**收藏分享梦境**（在浏览时可进行这些操作）

### 前置依赖

- **获取AI解梦**、**相似梦境匹配**、**分享梦境** 均需用户已有梦境记录（前置条件）
