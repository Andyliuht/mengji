# 梦境记录与分享平台 - 软件工程课程项目

> 一款帮助用户记录梦境、AI解梦、相似梦境匹配、匿名分享的小众应用

## 项目概述

- **项目名称**：梦境记录与分享平台
- **课程**：软件工程
- **团队**：3-5人

## 项目结构

```
软件工程/
├── README.md                 # 项目说明（本文件）
├── 项目指导.md               # 分步指导文档
├── 01-requirements/          # 第一步：项目立项与需求确认
├── 02-use-case-model/        # 作业1：用例模型（35%）
├── 03-design-model/          # 作业2：设计模型（35%）
│   ├── 静态结构设计/         # 类图等（10%）
│   └── 动态结构设计/         # 时序图、活动图等（25%）
└── 04-system/                # 作业3：系统验收（30%）
```

## 时间线

| 作业 | 内容 | 成绩 | 时间 |
|------|------|------|------|
| 1 | 用例模型 | 35% | 第4周公布，第6周提交 |
| 2 | 设计模型（动态25% + 静态10%） | 35% | 第8周公布，第10周提交 |
| 3 | 系统验收 | 30% | 第12周、第14周验收 |
