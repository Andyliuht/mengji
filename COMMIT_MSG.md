# 提交信息（Commit Message）规范

## 格式

```
<类型>: <简短描述>
```

## 类型说明

| 类型 | 说明 | 示例 |
|------|------|------|
| feat | 新功能 | feat: 添加梦境导出功能 |
| fix | 修复问题 | fix: 修复登录后跳转失败 |
| docs | 文档更新 | docs: 更新部署指南 |
| style | 代码格式（不影响逻辑） | style: 统一缩进 |
| refactor | 重构 | refactor: 优化 API 请求逻辑 |
| chore | 构建/工具/配置 | chore: 添加 04-system 英文路径用于部署 |

## 示例

```
chore: 添加 04-system 目录以支持 Render 部署
feat: 实现 AI 解梦功能
fix: 修复相似梦境匹配为空的问题
docs: 完善详细部署指南
```
