# 让其他设备访问网站 - 分步指南

---

## 第一步：启动后端服务

1. 打开 **终端/命令行**（VS Code 里按 `` Ctrl+` `` 或菜单 终端 → 新建终端）
2. 输入以下命令并回车：

```
cd d:\vscode\软件工程\04-系统实现\backend
```

3. 再输入：

```
npm run dev
```

4. 看到 `后端服务运行在 http://localhost:3000` 表示成功
5. **不要关闭这个终端**，让它一直运行

---

## 第二步：启动前端服务

1. **再打开一个新的终端**（点击终端窗口的 + 号，或 终端 → 新建终端）
2. 输入：

```
cd d:\vscode\软件工程\04-系统实现\frontend
```

3. 再输入：

```
npm run dev
```

4. 看到类似 `Local: http://localhost:5173/` 和 `Network: http://192.168.x.x:5173/` 表示成功
5. **不要关闭这个终端**，让它一直运行

---

## 第三步：查看你的 IP 地址

在前端启动成功的输出里，找到 **Network** 那一行，例如：

```
Network: http://192.168.1.100:5173/
```

这里的 `192.168.1.100` 就是你的 IP 地址（你的可能不同）。

**如果没看到 Network 行**，在终端输入：

```
ipconfig
```

在输出中找到「以太网」或「WLAN」下的 **IPv4 地址**，例如 `192.168.1.100`。

---

## 第四步：记下访问地址

你的访问地址是：

```
http://你的IP地址:5173
```

例如：`http://192.168.1.100:5173`

---

## 第五步：用其他设备访问

1. 确保**手机/另一台电脑**和你的电脑连接的是**同一个 WiFi**
2. 在手机或另一台电脑的浏览器里输入：`http://你的IP:5173`
3. 打开后应能看到登录页面

---

## 第六步：注册和登录

1. 点击「还没有账号？立即注册」
2. 输入用户名、密码，点击注册
3. 注册成功后会自动进入首页
4. 之后可以用该账号登录

---

## 常见问题

**Q：其他设备打不开网页？**
- 检查是否同一 WiFi
- 检查 Windows 防火墙：设置 → 防火墙 → 允许应用通过防火墙 → 勾选 Node.js 或添加 5173、3000 端口

**Q：本机可以打开，其他设备不行？**
- 可能是防火墙拦截，尝试暂时关闭防火墙测试
