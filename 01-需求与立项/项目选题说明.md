# 梦境记录与分享平台 - 项目选题说明

> 用于提交给老师确认的选题文档

---

## 一、项目名称

**梦境记录与分享平台**（Dream Journal & Share Platform）

*备选名称：梦迹、梦语、梦境树洞*

---

## 二、项目创意与新颖性

### 2.1 创意点

1. **小众垂直领域**：专注梦境记录与解读，市面上同类产品较少
2. **AI 解梦**：结合关键词、主题分析，为用户提供梦境解读参考
3. **相似梦境匹配**：基于标签、关键词匹配，发现做过类似梦的用户，形成共鸣
4. **匿名分享**：梦境内容私密，支持匿名分享与社区互动

### 2.2 与现有产品的差异

| 对比维度 | 普通日记/笔记 App | 本产品 |
|---------|-------------------|--------|
| 内容类型 | 日常记录 | 梦境专属 |
| 核心功能 | 增删改查 | 记录 + AI解梦 + 相似匹配 + 分享 |
| 社交属性 | 弱/无 | 匿名分享、相似梦境发现 |
| 智能分析 | 无 | AI解梦、个人梦境统计 |

---

## 三、功能需求清单（≥10个，不含注册登录）

根据课程要求，功能点至少 10 个且不能仅限增删改查。以下为初步功能清单：

| 序号 | 功能名称 | 功能描述 | 是否超出 CRUD |
|------|----------|----------|---------------|
| 1 | 梦境记录 | 记录梦境内容（文字/语音/图片），支持标签、情绪、睡眠质量 | 是（多媒体、结构化） |
| 2 | 梦境标签管理 | 自定义标签、系统推荐标签、标签云 | 是 |
| 3 | AI 解梦 | 基于关键词或规则对梦境进行解读 | 是 |
| 4 | 相似梦境匹配 | 根据标签、关键词匹配相似梦境 | 是 |
| 5 | 梦境分享 | 匿名/公开分享到社区 | 是 |
| 6 | 梦境评论与互动 | 对分享的梦境进行评论、点赞、收藏 | 是 |
| 7 | 梦境发现 | 热门梦境、话题、按标签/情绪筛选 | 是 |
| 8 | 梦境日历/时间线 | 按时间浏览个人梦境记录 | 是 |
| 9 | 个人梦境统计 | 梦境主题分布、情绪趋势、记录频率等可视化 | 是 |
| 10 | 梦境导出 | 导出个人梦境记录（PDF/文本） | 是 |
| 11 | 记录提醒 | 晨间提醒用户记录梦境 | 是 |
| 12 | 解梦收藏 | 收藏 AI 解梦结果 | 是 |

**说明**：以上功能均超出简单增删改查，涉及智能分析、匹配算法、社交互动、统计可视化等。

---

## 四、技术实现要求

- **面向对象**：采用 OOP 设计，模块化开发
- **技术栈建议**：Web 端（Vue/React + 后端 API）或 移动端（Flutter/原生）
- **AI 解梦**：初期可采用规则+关键词实现，后续可扩展接入大模型 API

---

## 五、实际应用背景

1. **心理学与自我探索**：梦境记录有助于自我认知与情绪梳理
2. **创意灵感**：许多创作者从梦境中获取灵感
3. **睡眠与健康**：结合睡眠质量记录，辅助睡眠改善
4. **社区共鸣**：相似梦境匹配带来归属感与趣味性
