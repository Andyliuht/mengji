# 用例图说明

## 一、用例图文件

- **PlantUML 源文件**：`03-用例图.puml`
- **查看方式**：
  - 使用 [PlantUML 在线编辑器](https://www.plantuml.com/plantuml/uml/) 粘贴代码生成图片
  - 在 VS Code 中安装 PlantUML 插件预览
  - 使用 draw.io、StarUML、Visual Paradigm 等工具根据下方说明重绘

## 二、用例图结构说明

### 2.1 参与者

| 参与者 | 关联用例 |
|--------|----------|
| **访客** | 用户注册、用户登录、浏览分享梦境、梦境发现 |
| **注册用户** | 除管理功能外的全部用例 |
| **管理员** | 内容审核、用户管理、浏览分享梦境、梦境发现 |

### 2.2 用例分组

| 分组 | 用例 |
|------|------|
| 账户 | 用户注册、用户登录 |
| 梦境记录 | 记录梦境、编辑梦境、删除梦境、管理梦境标签 |
| AI与匹配 | 获取AI解梦、收藏解梦结果、相似梦境匹配 |
| 分享与互动 | 分享梦境、浏览分享梦境、评论、点赞、收藏分享 |
| 发现与浏览 | 梦境发现、浏览梦境日历、浏览梦境时间线 |
| 统计与导出 | 查看个人梦境统计、导出梦境、设置记录提醒 |
| 管理 | 内容审核、用户管理 |

### 2.3 用例关系

**包含（Include）**：
- 记录梦境 → 管理梦境标签（记录时需选择/添加标签）
- 获取AI解梦、相似梦境匹配、分享梦境 → 记录梦境（需基于已有梦境）

**扩展（Extend）**：
- 浏览分享梦境 ← 评论梦境、点赞梦境、收藏分享梦境（浏览时可进行这些操作）

## 三、导出为图片

若使用 PlantUML 命令行：

```bash
java -jar plantuml.jar 03-用例图.puml -tpng
```

将生成 `03-用例图.png` 图片文件，可插入到作业文档中。
